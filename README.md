## Browser Extension template
- Template for creating simple firefox and chrome extensions.
- Chrome only accept manifest version 3, while Firefox still accept manifest version 2.
- There are two extension examples:
    - beastify: On the `beastify` directory, to create extension with popup that can replace content of active tab with beast images.
    - borderify: On the `borderify` directory, to create red border around firefox website.
